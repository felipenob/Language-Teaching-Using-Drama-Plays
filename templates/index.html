Language Teaching using Shakespeare Plays

Project created by Felipe Nobrega
Contact: email felipeaugnobrega@gmail.com / github felipenob
Current version: 1.1

Goal
Provide easy access to grammar topics that are present on William Shakespeare drama plays and may be useful for language teachers. Therefore, the project can be used to facilitate the usage of literature in the classroom and shorten class preparation time.

Currently available grammar topics:
Modal verbs
(more grammar topics will be added in the future)

Instructions:
1. Select a Shakespeare play from the list.
2. Choose a modal verb.







Attention!
The modal verbs are classified by lemma. It means that, for instance, in the group can you can find all related modal verbs such as cannot, cannst etc.

In order to help you, we provide a list with all modal verbs by group.
